<h1>Λειτουργικές Απαιτήσεις</h1>
	<ul>
				<li>Να αποθηκευει τα στοιχεια του ζώου προς υιοθέτηση</li>
				<li>Να επιτρεπει την αναζητηση ζώου απο τη βάση</li>
				<li>Εξαγωγή στατιστηκών</li>
				<li>Διαχείρηση αίτησης</li>
		
	</ul>
	
	<h1>Μη Λειτουργικές Απαιτήσεις</h1>
	<ul>
				<li>Μόνο ο διαχειριστής μπορεί να εχει πρόσβαση στην
πλατφορμα του API</li>
				<li>Για την τελική απόφαση το συστημα χρειάζεται μεχρι και 
5 εργάσιμες</li>
				<li>Τα στοιχεία του φιλόζωου και της φιλοζωικής
πρέπει να προστατεύονται</li>
				<li>Ο φιλοζωος θα πρεπει να μπορει να 
κάνει αναζητηση βαση των queries</li>
		
	</ul>
	
	<h1>Βασική Ροή</h1>
	
	<ol>
		<li>Ο φιλόζωος κάνει είσοδο στο σύστημα</li>
		<li>Ο φιλόζωος κάνει αναζήτηση κατοικιδίου </li>
		<li>Το σύστημα παρουσιάζει ζώα διαθλεσιμα προς υιοθεσία (3α το σύστημα δεν έχει διαθέσιμα ζώα)</li>
		<li>Ο φιλόζωος επιλέγει κατοίκιδιο</li>
		<li>Το σύστημα αποδέχεται την αίτηση</li>
		<li>Το σύστημα μεταβιβάζει την αίτηση στον υπάλληλο φιλοζωικής</li>
		<li>Ο υπάλληλος διαχειρίζεται την αίτηση(7α απόρριψη αίτησης)</li>
		<li>Ο υπάλληλος προωθεί την απάντηση στον πελάτη </li>
		<li>Ο διαχειριστής εξάγει στατιστηκά στοίχεια</li>
	</ol>
