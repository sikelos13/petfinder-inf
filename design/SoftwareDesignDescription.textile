<p><strong>Interface UserAccount </strong>
<br />Η διεπαφή δηλώνει μία function που απαιτείται για το login των χρηστών του συστήματος.</p>

<p><strong>Class Ad </strong>
<br />Κλάση που περιγράφει την αγγελία ενός ζώου προς υιοθεσία. Εκτός των functions get/set υπάρχει η setActive και η isActive function που δηλώνει αν μια αγγελία είναι ενεργή ή έχει κλείσει (δηλαδή έχει γίνει η υιοθεσία).</p>

<p><strong>Class Adoption </strong>
<br />Κλάση που περιγράφει την υιοθεσία ενός ζώου. Στη κλάση γίνεται ο ορισμός του Applicant όπου η function setApplicant δηλώνει το ιδιοκτήτη του ζώου και καλεί τη PetAdoption του Applicant η οποία ολοκληρώνει την διαδικασία της υιοθεσίας.</p>

<p><strong>Class AnimalWelfare (κληρονομεί την class user) </strong>
<br />Κλάση που περιγράφει μία φιλοζωική οργάνωση. Στη κλάση δηλώνονται τα στοιχεία της φιλοζωικής.  Τα instances της κλάσης χρησιμοποιούνται ως author σε instances της κλάσης Ad (αγγελία).</p>

<p><strong>Class Applicant (κληρονομεί την class user) </strong>
<br />Κλάση που περιγράφει έναν υποψήφιο ιδιοκτήτη ζώου. Στη κλάση δηλώνονται τα στοιχεία του υποψηφίου. Η function PetAdoption προσθέτει νέο ζώο στη λίστα του Applicant και θέτει τo ζώο ως υιοθετημένο.</p>

<p><strong>Class Breed </strong>
<br />Κλάση που περιγράφει την ράτσα ενός ζώου. Στη κλάση δηλώνονται τα στοιχεία της ράτσας. Μέσα στη κλάση υπάρχει ένα List Vector με τα instances των Breeds τα οποία είναι μη συμβατά. Η λίστα αυτή χρησιμοποιείται για τον έλεγχο της υιοθεσίας ενός ζώου, αν η ράτσα του είναι συμβατή με τα υπόλοιπα ζώα που έχει στην κατοχή ο Applicant.</p>

<p><strong>Class Employee (κληρονομεί την class user) </strong>
<br />Κλάση που περιγράφει τον διαχειριστή χρήστη του συστήματος όπου καταχωρεί τα στοιχεία των ρατσών. Το instance εγκρίνει ή όχι μία υιοθεσία μέσω της function acceptAdoption.</p>

<p><strong>Class Image </strong>
<br />Κλάση που περιγράφει ένα instance μιας εικόνας (url και περιγραφή).</p>

<p><strong>Class Pet </strong>
<br />Κλάση που περιγράφει ένα ζώο. Μέσω της function hasBeenAdopted μπορούμε να ελέγξουμε αν το ζώο έχει υιοθετηθεί.</p>

<p><strong>Class User </strong>
<br />Υπερκλάση που περιγράφει έναν χρήστη του συστήματος με τα βασικά του στοιχεία.</p>
